주제: GCN 임베딩 및 Fingerprint를 결합한 Two-Stream Stacking 앙상블

1. 모델 아키텍처 다이어그램

본 모델은 데이터의 특성에 따라 두 개의 독립적인 트랙(Track)을 운영하며, 최종적으로 메타 학습기(Meta-Learner)를 통해 결과를 도출합니다.

[Track 1: Graph Path]
GCN Embeddings → StandardScaler → LR + SVM (Voting) → P1 (Proba)
[Track 2: Tabular Path]
ECFP Fingerprints → RF + XGB + LGBM (Voting) → P2 (Proba)
[Final Stage: Stacking]
Concatenate [P1, P2] → Meta-Learner (Logistic Regression) → Final Prediction

2. 주요 설계 전략

  • 데이터 맞춤형 모델링: 연속형 고차원 데이터인 GCN 임베딩에는 선형 모델(LR, SVM)을, 이진 데이터인 Fingerprint에는 트리 모델(RF, XGB, LGBM)을 배치하여 성능을 극대화했습니다.
  • Stacking Ensemble: 단순 평균(Soft Voting) 대신 Logistic Regression을 메타 학습기로 사용하여, 검증 데이터에서 각 트랙의 신뢰도(가중치)를 스스로 학습하게 설계되었습니다.
  • Threshold Optimization: MCC(Matthews Correlation Coefficient)를 최대화하는 임계값을 0.1~0.9 사이에서 탐색하여 불균형 데이터셋에 대응합니다.

심판(Meta-Learner)의 역할: meta_model.coef_를 통해 Track 1과 Track 2 중 어떤 데이터가 예측에 더 기여했는지 파악합니다. 예를 들어 Track 2의 가중치가 높다면, 해당 화합물은 구조적 지문(Fingerprint) 정보가 더 유효하다는 뜻입니다.

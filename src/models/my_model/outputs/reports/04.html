2. 현행 아키텍처의 화학 구조 Topology(위상) 보존 여부 통찰

진단: 위상학적 선후 연결(Topology) 정보는 완전히 파손(Broken)되었습니다. 현재 어텐션 모델에 피딩되는 Local Feature 서브 인코더는 BRICS 화학 가위에 의해 분자를 분리해 낸 후 단순히 조각들의 "출현 빈도"와 "단어 사전(Vocabulary)" 위상을 매핑하는 Bag-of-Fragments (Bag-of-Words)에 전적으로 의존하고 있습니다.
  • 단어 분절화의 한계: 벤젠 고리(Fragment A)와 아민기(Fragment B)가 결합해 있든, 멀리 떨어져 있든 현재 전처리 행렬 공간에선 단순히 [A: 1, B: 1]...로 벡터 카운팅이 될 뿐입니다. 즉, "어느 원자와 어느 원자가 단일 결합인지, 이중 결합인지"에 대한 화학적 그래프 구조(Graph Topology) 맥락이 증발합니다.
  • Morgan 피처가 보충? 한계: 같이 입력되는 256차원 글로벌 Morgan FP는 주변 거리에 따른 해시(Hash) 식별자를 제공하므로 어느 정도 보완이 되나, 이것은 존재의 조합적 지문일 뿐 공간 내의 이웃 전파(Message Passing)를 물리적으로 추적하고 내포하지 못합니다.

3. Topology 훼손 솔루션: 향후 개선 모델 (1-Layer GCN 기반 서브 분자 인코더 도입)

Bag-of-Fragments의 근시안적인 빈도 카운팅을 버리고, 원자 수준(Atom-level) 연결 위상을 100% 보존하면서 Toxicophore(독성 작용기)를 잡아내는 GCN(Graph Convolutional Network) 어텐션 하이브리드 아키텍처를 도입하는 것이 차세대 DILI 예측 모델의 정답(Silver Bullet)입니다.

1) 왜 GCN을 그동안 꺼려왔고, 왜 하필 1-Layer인가?

  • Oversmoothing (신호 뭉개짐 현상): GCN 레이어를 3계층~5계층으로 일반 딥러닝처럼 깊이 연속할 경우, 화학식 양 끝단의 원자 정보가 계속 이웃의 이웃으로 섞여 들어가며(Message Passing), 분자 내 모든 노드(원자)의 벡터 색깔이 비슷해지는 심각한 부작용이 나타납니다.
  • 해결책: DILI를 유발하는 작용기(Toxicophore)는 보통 반경(Radius) 1~2 안의 이웃과만 상호작용하는 아민이나 황 결합 등 국소적(Local) 형태를 가집니다. 따라서 딱 반경 1 이웃들의 결합 구조(단일/이중 등 Edge 특징)만 1번 수용하는 1-Layer GCN을 배수진으로 배치합니다. 위상은 유지하면서 노드 고유의 날카로운 독성 신호(Specificity)는 잃지 않게 됩니다.

2) Crossed Differential Attention 과 GCN의 완벽한 궁합

  • 기존의 sub_train, sub_test (BoW 입력 타겟 통짜 피처) 자리에 GCN을 통과한 원자 수 N개의 로컬 Node Feature Matrix (N x Dim)을 거름망 없이 그대로 던집니다.
  • 기존의 메인 블록이었던 Differential Attention 유닛이 [Query] 로 글로벌 Morgan 지문을 쥐고, GCN Node 행렬 [Key, Value] 속의 모든 원자(Atom)들을 하나씩 순회하며 노이즈 신호를 제거(A1 - λA2 계산)하고 독성 반응기가 숨어있는 정확한 원자 이웃을 역추적해 강한 Attention 가중치를 찍게 됩니다.

3) 차세대 XAI (설명 가능성)의 실현

GCN-Attention 이 결합하면 이제 더 이상 "이 단어 조각의 점수가 높습니다" 식의 표면적 해석에서 안주하지 않아도 됩니다. 어텐션 스코어가 분자 그래프를 이루는 N개의 실제 원자 좌표 위(2D Molecule Drawing)에 빨간색 활성도(Heat)로 그림핑 될 수 있으며, "간독성의 원인이 이 방향의 벤젠 치환기에 집중되어 있다"는 논문급의 XAI 도출이 가능해집니다.

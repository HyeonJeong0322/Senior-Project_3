3. 논문 작성용 아키텍처 흐름 설명 요약

  • 입력 분리 (Feature Decoupling): SMILES 열은 글로벌 지문(FP) 분리 추출과 GCN 모듈을 위한 분자 그래프 구조 생성 두 갈래로 나뉩니다. 이로써 전역적 구조 정보와 국소 위상 정보의 원본을 각각 확보합니다.
  • Graph Topology 보존 인코딩 (1-Layer GCN): 기존 BoF(Bag-of-Fragments)의 위상 훼손 맹점을 개선하여, 1-Hop Message Passing에 국한시킨 GCN을 통해 극심한 흔들림(Oversmoothing) 없이 독성 발현의 주 원인인 국소 이웃 결합구조를 온전히 Node 매트릭스(K, V)로 수렴시킵니다.
  • Crossed Differential Attention: 글로벌 맥락 정보(Q)가 국소 Node 매트릭스를 순회하며 각 원자 이웃의 '독성 연관도'를 판별합니다. 이때 A_diff = A1 - λ·A2 구조로 이루어진 차등 연산을 통해 허위 신호(Noise)는 수학적으로 상쇄(Cancel)시키고 Toxicophore(독성 작용기) 원자들의 정보만 극적으로 집중시킵니다.
  • Feature Extraction (Projection Bypass): GCN-Attention Readout으로 얻어진 차원 축소된 국소 성분과 전역 FP 성분을 병합(Concat)합니다. 학습 중에만 Loss 전파를 돕던 보조 기구(Auxiliary Head)는 제거되고, 밀도 높은 128-차원 임베딩 행렬 특징만 최종 산출물로 배출합니다.
